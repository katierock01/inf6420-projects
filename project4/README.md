# Project 4 - Responsive Squirrels Website

Responsive redesign of Project 3 with mobile-first design and media queries.

## Features
- Responsive navigation
- Flexible layouts (flexbox/grid)
- Mobile, tablet, and desktop breakpoints
- Touch-friendly interface
- Optimized images

## Pages
- home.html - Landing page
- fox.html, gray.html, red.html, flying.html - Species pages

## Stylesheet
- squirrels-responsive.css - Responsive external stylesheet
